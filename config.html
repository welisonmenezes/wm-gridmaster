WM GridMaster - Plugin jQuery para galeria de conteúdo estilo portfólio.

Opções de configuração do plugin WM GridMaster.

cols

(int) - Seta o número de colunas que o grid terá. Como veremos também é possível trabalhar com responsividade.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										cols : 8,
									});
close_button_text

(string) - O texto do botão de fechar o detalhe.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										close_button_text : 'X'
									});
next_button_text

(string) - O texto do botão de próximo detalhe.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										next_button_text : 'next'
									});
prev_button_text

(string) - O texto do botão de detalhe anterior.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										prev_button_text : 'prev'
									});
go_to_type

(string) - onde deve parar o scroll após clicar em algum thumb. Opções: [none, top-thumb, middle-thumb, top-content], onde:
none = não rola scroll.
top-thumb = rola para o topo do thumb.
middle-thumb = rola para o meio do thumb.
top-content = rola para o topo do conteúdo do detalhe.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										go_to_type : 'middle-thumb'
									});
has_detail

(boolean) - define se o thumb terá detalhe ou não.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										has_detail : true
									});
has_navigation

(boolean) - define se o detalhe terá navegação de next/prev.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										has_navigation : true
									});
navigation_position

(string) - Seta a posição vertical dos botões de navegação dentro do detalhe. Opções: [none, top, middle, bottom], onde:
none = não será feito nenhum posicionamento.
top = posiciona no topo do detalhe.
middle = posiciona no meio do detalhe.
bottom = posiciona no bottom do detalhe.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										navigation_position : 'middle'
									});
thumbs_height

(string or int) - seta a altura dos thumbnails. Opções: [equal-width, row-equal-width, int], onde:
equal-width = manterá a altura proporcional a largura automaticamente.
row-equal-width = manterá a altura dos elementos de mesma linha igual ao elemento de maior altura.
int = o valor em pixel da altura que deseja.
Exemplo: 

									$('.wm-grid').wmGridMaster({
										thumbs_height : 'equal-width'
									});
spaces

(object js) Objeto para configurar o espaçamento entre os thumbnails e o padding do wrap do grid

padding

(int) O espaçamento entre os thumbinail.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												spaces : {
													padding : 5
												},
											});
wrap

(boolean) Indica se o wrpa do grid terá um padding. A largura do mesmo será o valor definido em spaces.padding.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												spaces : {
													wrap : true
												},
											});
arrow_detail

(object js) Objeto para configurar a seta que indica a posição do conteúdo aberto em relação ao seu thumbinail.

has_arrow

(boolean) Determina se terá a seta ou não.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												arrow_detail : {
													has_arrow : true
												},
											});
arrow_size

(int) Seta a largura da seta em pixels.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												arrow_detail : {
													arrow_size : 10
												},
											});
speeds

(object js) Objeto para configurar as velocidades das animações do grid.

content_down

(int) Determina a velociade do slideDown da visualização do detalhe do thumbnail.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												speeds : {
													content_down : 300
												},
											});
content_up

(int) Determina a velociade do slideUp da visualização do detalhe do thumbnail.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												speeds : {
													content_up : 300
												},
											});
go_to_opened

(int) Determina a velociade do scroll até o thumbnail selecionado. Terá efeito caso go_to_type seja diferente de 'none'.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												speeds : {
													go_to_opened : 300
												},
											});
ajax

(object js) Objeto para configurar as requisiçoes ajax do grid.

has_ajax

(boolean) Indica se usará a função padrão do grid para carregamento de mais thumbnails via ajax.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													has_ajax : true
												},
											});
method

(string) Seta o tipo da requisição do carregamento de mais thumbnails. Opções: [GET, POST].
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													method : 'GET'
												},
											});
data

(object js) Objeto com os dados a serem enviados na requisição de mais thumbnails.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													data : { page : 2, filter : 'a' }
												},
											});
url

(string) Seta a Url para a requisição do carregamento de mais thumbnails.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													url : 'http://domain.com.br'
												},
											});
button_text

(string) Seta o texto do botão gerado pelo grid para o carregamento de mais thumbnails.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													button_text : 'Load more'
												},
											});
load_content

(string) Seta o texto ou html do loading gerado pelo grid para o carregamento de mais thumbnails. Será exibido durante o carregamento ajax.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													load_content : 'carregando...'
												},
											});
has_ajax_detail

(boolean) Determina se o carregamento do detalhe será feito via Ajax.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													has_ajax_detail : true
												},
											});
method_detail

(string) Seta o tipo da requisição do carregamento do detalhe. Opções: [GET, POST].
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													method_detail : 'GET'
												},
											});
data_detail

(object js) Objeto com os dados a serem enviados na requisição do detalhe.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													data_detail : { id : 2 }
												},
											});
url_detail

(string) Seta a Url para a requisição do detalhe.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													url_detail : 'http://domain.com.br'
												},
											});
load_content_detail

(string) Seta o texto ou html do loading gerado pelo grid para o carregamento do detalhe. Será exibido durante o carregamento ajax.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												ajax : {
													load_content_detail : 'Carregando...'
												},
											});
selectors

(object js) Objeto para customizar os seletores do plugin (Apenas classes css.). OBS: Precisam estar de acordo com os que você setou no HTML.

grid_has_opened

(string) Classe adicionada dinamicamente no grid quando existe algum conteúdo aberto.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													grid_has_opened : 'wm-grid-has-opened'
												},
											});
wrap

(string) Classe do wrap dos itens.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													wrap : 'wm-grid-wrap'
												},
											});
item

(string) Classe dos itens.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													item : 'wm-grid-item'
												},
											});
thumbs

(string) Classe do container do thumbnail.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													thumbs : 'wm-grid-thumb'
												},
											});
item_opened

(string) Classe adicionada dinamicamente no item aberto para visualização do seu detalhe.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													item_opened : 'wm-grid-item-opened'
												},
											});
items_last_row

(string) Classe adicionada dinamicamente nos itens da última linha.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													items_last_row : 'wm-grid-of-last-row'
												},
											});
content

(string) Classe do wrap do detalhe.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													content : 'wm-grid-content'
												},
											});
body_content

(string) Classe do container do conteúdo de detalhe.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													body_content : 'wm-grid-body'
												},
											});
button_close

(string) Classe adicionada dinamicamente no botão de fechar o detalhe gerado pelo grid.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												selectors : {
													button_close : 'wm-grid-close'
												},
											});
button_next

(string) Classe adicionada dinamicamente no botão de próximo detalhe gerado pelo grid.
Exemplo: 

											$('.wm-grid').wmGridMaster({

Exemplo:

callbacks

(object js) Objeto para configurar os callbacks do grid.

CB_beforeInitGrid

(function) Executado antes da inicialização completa do grid.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												callbacks : {
													CB_beforeInitGrid : function(){
														console.log('CB_beforeInitGrid');
													}
												},
											});
CB_afterInitGrid

(function) Executado depois da inicialização completa do grid.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												callbacks : {
													CB_afterInitGrid : function(){
														console.log('CB_afterInitGrid');
													}
												},
											});
CB_beforeLoadAjax

(function) Executado antes de fazer a requisição de mais thumbnails. Só roda quando ajax.has_ajax está definido como true.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												callbacks : {
													CB_beforeLoadAjax : function(){
														console.log('CB_beforeLoadAjax');
													}
												},
											});
CB_afterLoadAjax

(function) Executado depois que a requisição ajax de mais thumbnails responde. Só roda quando ajax.has_ajax está definido como true.
Exemplo: 

											$('.wm-grid').wmGridMaster({
												callbacks : {

													}
												},

CB_beforeOpenDetail

responsive

(array) Um array de objetos que configura algumas opções para a resolução definida pelo usuário.

size:

Aqui se define a resolução cujas configurações a seguir, em options, se aplicarão.
Exemplo: 

												$('.wm-grid').wmGridMaster({
													responsive : [
														{
															size : 992,
															options : {}
														}
													],
												});
options:

Onde se ajusta algumas configurações globais para se aplicarem na resolução definia em size. Atenção: Apenas as opções mostradas no exemplo abaixo podem ser customizadas em responsive.
Exemplo: 

												$('.wm-grid').wmGridMaster({
													responsive : [
														{
															size : 992,
															options : {
																cols : 6, 
																spaces : {
																	padding : 3,
																	wrap : true
																},
																thumbs_height : 'equal-width'
															}
														}
													],
												});

Funções da Api do Grid

Algumas funções do Grid podem ser acessadas externamente.


Função init()

Chamada para inicializar o Grid. Também é ideal para ser chamada após uma chamada Ajax feita de maneira customizada.
Exemplo: 

							var myGrid = $('.wm-grid').wmGridMaster({});
							myGrid.init();

							$('.add-more').on('click', function(e){
								$.ajax({
									method: "GET",
									url: "ajax.php",
									data: {}
								})
								.done(function(msg) {

									// note que o .wm-grid-wrap é o wrap dos itens do grid
									$('.wm-grid .wm-grid-wrap').append(msg);
									myGrid.init();

								});

								e.preventDefault();
								return false;
							});


Função goTo()

Usada para abrir o detalhe de um item específico passado por parâmetro. São necessário 2 parâmetros. o 1º é o Grid, e o segundo o index do elemento a ser aberto. (o index inicia do zero).
Exemplo: 

							var myGrid = $('.wm-grid').wmGridMaster({});
							myGrid.init();

							$('.go-to-6').on('click', function(){

								myGrid.goTo($('.wm-grid'), 6);

								return false;
							});


Função closeDetail()

Usada para fechar o detalhe aberto do Grid. O parâmetro requerido é o Grid.
Exemplo: 

							var myGrid = $('.wm-grid').wmGridMaster({});
							myGrid.init();

							$('.close-detail').on('click', function(){

								myGrid.closeDetail($('.wm-grid'));

								return false;
							});
